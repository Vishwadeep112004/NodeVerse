What is BFS?

Breadth First Search explores all neighbors of a node before moving to the next level. Uses a Queue (FIFO) structure.

How It Works

  • Start from source node, mark visited
  • Push into queue
  • While queue not empty: dequeue, visit all unvisited neighbors
  • Mark neighbors visited, push to queue

Complexity

  • Time: O(V + E)
  • Space: O(V)

Applications

  • Shortest path in unweighted graphs
  • Level order traversal
  • Connected components
  • Bipartite graph detection
