What is Depth First Search (DFS)?

Depth First Search (DFS) is a graph traversal algorithm that explores as far as possible along a branch before backtracking. It uses a Stack (LIFO) structure, either explicitly or via recursion.

How DFS Works

  • Start from a source node.
  • Mark it as visited.
  • Explore one of its unvisited neighbors.
  • Repeat the process recursively for that neighbor.
  • If no unvisited neighbors remain, backtrack.
  • Continue until all nodes are visited.

Key Properties of DFS

  • Uses Stack (or recursion).
  • Goes deep before exploring breadth.
  • Does NOT guarantee shortest path.
  • Works on both directed and undirected graphs.

Time & Space Complexity

  • Time Complexity: O(V + E)
  • Space Complexity: O(V)

Applications of DFS

  • Detecting cycles in graphs.
  • Topological sorting (DAG).
  • Finding connected components.
  • Solving puzzles (maze, backtracking).
  • Strongly Connected Components (Kosaraju, Tarjan).
  • Path finding in games and AI.

Why DFS is Important?

DFS is useful when exploring all possible paths or solutions, especially in problems involving recursion, backtracking, and graph structure analysis.
