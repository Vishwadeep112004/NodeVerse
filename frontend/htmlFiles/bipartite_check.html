What is a Bipartite Graph?

A graph is bipartite if its vertices can be divided into two disjoint sets such that every edge connects a vertex from one set to the other. Equivalently, a graph is bipartite if and only if it contains no odd-length cycle.

BFS 2-Coloring Method

  • Assign color 0 to start node (Blue).
  • For each unvisited neighbor: assign opposite color.
  • If a neighbor has the same color → NOT bipartite.
  • Handles disconnected graphs automatically.

Color Legend

  • Blue — Side A (color 0)
  • Orange — Side B (color 1)
  • Red — Conflict! (same color neighbors)

Time & Space Complexity

  • Time: O(V + E)
  • Space: O(V)

Applications

  • Matching problems (job assignment).
  • Detecting even/odd cycles.
  • Network flow problems.
  • Stable marriage problem.
