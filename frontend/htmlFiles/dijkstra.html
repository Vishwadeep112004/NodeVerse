What is Dijkstra's Algorithm?

Dijkstra's algorithm finds the shortest path from a source node to all other nodes in a weighted graph with non-negative edge weights. It uses a greedy approach with a min-priority queue.

How it Works

  • Set dist[src] = 0, all others = ∞.
  • Push source into priority queue (min-heap by distance).
  • While queue is not empty:
  • → Extract node u with minimum dist.
  • → For each neighbor v with weight w:
  • → If dist[u] + w < dist[v], update dist[v].
  • → Push v into queue with new distance.

Color Legend

  • Yellow — current node (being processed)
  • Orange — distance just relaxed/updated
  • Green — node settled (shortest path found)
  • Label below node = current best distance

Warning️ Important

  • Weights must be entered in the Weights field.
  • Weights must align with edges by position.
  • Does NOT work with negative edge weights.

Time & Space Complexity

  • Time: O((V + E) log V) with min-heap
  • Space: O(V + E)

Applications

  • GPS navigation & Google Maps routing.
  • Network routing protocols (OSPF).
  • Shortest path in games & AI.
  • Airline & logistics route optimization.
