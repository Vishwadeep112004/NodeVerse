Topological Sort — DFS Based

What is Topological Sort?

A linear ordering of vertices of a DAG such that for every directed edge u → v, vertex u appears before v. Only valid on Directed Acyclic Graphs.

DFS Method

  • Run DFS from each unvisited node.
  • After fully exploring a node, push it onto a stack.
  • Reverse the stack to get topological order.
  • Key insight: a node is pushed only after all its descendants.

Color Legend

  • 🟡 Yellow — currently exploring (DFS active)
  • 🔵 Blue — backtracking from this node
  • 🟣 Purple — pushed onto result stack (finalized)

DFS vs Kahn's Algorithm

  • DFS: recursive, uses finish-order stack.
  • Kahn's: iterative, uses indegree queue (BFS-based).
  • Both produce valid topological orderings.
  • Kahn's can detect cycles; DFS needs extra tracking.

Time & Space Complexity

  • Time: O(V + E)
  • Space: O(V)
