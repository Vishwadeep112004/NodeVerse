What is a Minimum Spanning Tree?

A Minimum Spanning Tree (MST) of a weighted undirected graph is a spanning tree with the minimum possible total edge weight. It connects all vertices with no cycles and minimum cost.

Prim's Algorithm

  • Start from any node. Mark it as in MST.
  • Add the minimum weight edge connecting MST to non-MST vertex.
  • Repeat until all vertices are in MST.
  • Uses a min-priority queue (greedy approach).

Color Legend

  • 🟡 Yellow — start node selected
  • 🟠 Orange — candidate node (being considered)
  • 🟢 Green — node added to MST
  • Active edges — the MST edges chosen

Prim's vs Kruskal's

  • Prim's: grows MST from a source, good for dense graphs.
  • Kruskal's: sorts all edges, good for sparse graphs.
  • Both produce the same MST (may differ in edge choice for equal weights).

Time & Space Complexity

  • Time: O((V + E) log V) with min-heap
  • Space: O(V + E)

Applications

  • Network design (minimum cable/road cost).
  • Cluster analysis in machine learning.
  • Circuit design and VLSI layout.
  • Approximation algorithms for TSP.
