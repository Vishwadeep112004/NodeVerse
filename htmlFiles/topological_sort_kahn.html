Topological Sort — Kahn's Algorithm

What is Topological Sort?

A topological ordering of a Directed Acyclic Graph (DAG) is a linear arrangement of vertices such that for every directed edge u → v, vertex u appears before v in the ordering.

Kahn's Algorithm (BFS-based)

  • Compute indegree of all nodes.
  • Enqueue all nodes with indegree 0.
  • While queue is not empty:
  • → Dequeue node u, add to result.
  • → For each neighbor v: decrement indegree[v].
  • → If indegree[v] becomes 0, enqueue v.
  • If result size ≠ n → cycle detected.

Color Legend

  • 🟡 Yellow — in queue (indegree = 0)
  • 🟢 Green — currently being processed
  • 🟣 Purple — finalized in topo order

Time & Space Complexity

  • Time: O(V + E)
  • Space: O(V)

Applications

  • Build systems (Makefile dependency resolution).
  • Task scheduling with prerequisites.
  • Course prerequisite ordering.
  • Compiler instruction ordering.
  • Package managers (npm, pip).
