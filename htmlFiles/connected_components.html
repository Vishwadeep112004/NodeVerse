Connected Components Visualizer

What are Connected Components?

A connected component is a group of nodes in a graph where every node is reachable from every other node in the same group.

How It Works

  • Start from an unvisited node.
  • Perform BFS/DFS traversal.
  • Mark all reachable nodes as part of one component.
  • Repeat for all unvisited nodes.

Key Properties

  • Each node belongs to exactly one component.
  • Useful for detecting isolated subgraphs.
  • Works best with undirected graphs.

Time Complexity

  • O(V + E)

Applications

  • Social network analysis
  • Network connectivity
  • Image segmentation
  • Clustering problems
