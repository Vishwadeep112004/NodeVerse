Graph Algorithms

Basic Traversal

Beginner

Fundamental graph exploration strategies — the building blocks of all advanced algorithms.

BFS
Breadth First Search
Queue · O(V+E) · Shortest path in unweighted graphs
Beginner →
Explores level by level using a Queue. Finds shortest path in unweighted graphs.
DFS
Depth First Search
Stack · O(V+E) · Full graph exploration
Beginner →
Goes deep before backtracking. Uses Stack/recursion. Great for path finding.
CC
Connected Components
BFS/DFS · O(V+E) · Subgraph identification
Beginner →
Identifies all disconnected subgraphs using BFS/DFS from each unvisited node.
🔄

Cycle Detection

Intermediate

Detect cycles and verify graph properties like bipartiteness using coloring techniques.

CU
Cycle Detection (Undirected)
Union-Find / DFS · O(V+E)
Intermediate →
Detects back edges in an undirected graph using DFS with parent tracking.
CD
Cycle Detection (Directed)
DFS + Color · O(V+E)
Intermediate →
Uses 3-color DFS (white/grey/black) to detect back edges in a directed graph.
BP
Bipartite Graph Check
BFS 2-Coloring · O(V+E)
Intermediate →
Checks if a graph can be 2-colored (no odd cycles) using BFS with alternating colors.
📐

Topological Sort

Intermediate

Linear ordering of vertices in a DAG such that for every edge u→v, u comes before v.

TS
Topological Sort (DFS)
DFS + Stack · O(V+E) · DAG ordering
Intermediate →
Push nodes to a stack after completing all neighbors in DFS. Reverse gives topo order.
KN
Topological Sort (Kahn's)
BFS + In-Degree · O(V+E)
Intermediate →
BFS approach: repeatedly remove nodes with zero in-degree. Also detects cycles in DAGs.
🌀

Strongly Connected Components

Advanced

Find maximal subgraphs where every vertex is reachable from every other vertex.

KS
Kosaraju's Algorithm
2-Pass DFS · O(V+E) · SCC detection
Advanced →
Two DFS passes: record finish order, then DFS on reversed graph in reverse finish order.
TJ
Tarjan's Algorithm
DFS + Stack · O(V+E) · Single pass SCC
Advanced →
Single-pass DFS using low-link values and a stack. More elegant than Kosaraju's.
🗺️

Shortest Path

Advanced

Find minimum cost paths between vertices in weighted graphs.

DJ
Dijkstra's Algorithm
Priority Queue · O((V+E) log V) · Non-negative weights
Advanced →
Greedy algorithm using a min-heap. Only works with non-negative edge weights.
BF
Bellman-Ford Algorithm
Edge Relaxation · O(VE) · Negative edges OK
Advanced →
Relaxes all edges V-1 times. Handles negative weights and detects negative cycles.
FW
Floyd-Warshall Algorithm
DP · O(V³) · All-pairs shortest path
Advanced →
Dynamic programming to find shortest paths between ALL pairs of vertices.
🌲

Minimum Spanning Tree

Advanced

Find the minimum weight set of edges that connects all vertices without cycles.

PR
Prim's Algorithm
Greedy + PQ · O((V+E) log V) · Vertex-based
Advanced →
Grows MST from a starting vertex, always picking the minimum edge crossing the cut.
KR
Kruskal's Algorithm
Sort + Union-Find · O(E log E) · Edge-based
Advanced →
Sorts all edges by weight, adds each if it doesn't form a cycle (Union-Find).
