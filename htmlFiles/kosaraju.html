What are Strongly Connected Components?

A Strongly Connected Component (SCC) of a directed graph is a maximal set of vertices such that there is a path from every vertex to every other vertex within the set.

Kosaraju's Algorithm — 2 DFS Passes

  • Pass 1: Run DFS on original graph.
  • → Record nodes in finish order onto a stack.
  • Pass 2: Reverse all edges to get transpose graph.
  • → Pop from stack and run DFS on reversed graph.
  • → Each DFS tree in pass 2 = one SCC.

Color Legend

  • 🟡 Yellow — active in DFS (being explored)
  • 🟢 Green — visited in pass 1
  • ⬜ Slate — finished pass 1 (in stack)
  • 🟠 Orange — being explored in pass 2
  • 🎨 Final color = SCC group (each SCC gets unique color)

⚠️ Important

  • Works on directed graphs only.
  • Edge input format: 0-1, 1-2, 2-0, 3-0
  • Arrows in graph show direction.

Time & Space Complexity

  • Time: O(V + E) — two DFS passes
  • Space: O(V + E) — for reversed graph

Applications

  • Finding clusters in social networks.
  • Compiler optimization (basic blocks).
  • Solving 2-SAT problems.
  • Web crawling & link analysis.
  • Detecting cycles in directed graphs.
