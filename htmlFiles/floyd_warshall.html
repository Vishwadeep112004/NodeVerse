Floyd-Warshall Algorithm

What is Floyd-Warshall?

Floyd-Warshall finds shortest paths between all pairs of vertices in a weighted graph. It works with negative weights and detects negative cycles via diagonal checks.

How it Works (3 nested loops)

  • For each intermediate node k (0 to n-1):
  • → For each pair (i, j):
  • → If dist[i][k] + dist[k][j] < dist[i][j]:
  • → Update dist[i][j] = dist[i][k] + dist[k][j]
  • If dist[i][i] < 0 after → negative cycle.

Visual Output

  • 🟡 Yellow node — current intermediate node k
  • 🟠 Orange nodes — pair (i,j) being updated
  • 📊 Distance matrix shown live in code panel
  • 🟢 Green all nodes — algorithm complete

⚠️ Note

  • Keep nodes ≤ 6 for readable matrix display.
  • Weights are required for this algorithm.

Time & Space Complexity

  • Time: O(V³)
  • Space: O(V²) — the distance matrix

Applications

  • All-pairs shortest paths in road networks.
  • Transitive closure of a graph.
  • Network latency analysis.
